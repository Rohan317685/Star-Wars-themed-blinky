# Star-Wars-themed-blinky
This is a Star Wars themed 555 Blinky made for HackClub!

Hack Club is a 501(c)(3) non-profit that helps teens make things. For more info, go to [hackclub.com](https://hackclub.com).

This project PCB costs $3.50 and the customs and duties may defer depending on where you live!

To order the PCB go to JLPCB's official website which I find is the cheapest and most reliable company to order PCB's from!

Make an account and upload the zip files of the Gerber in the production files folder!

It should lead you to this page and it should show this after you uploaded! (It may be green you can change the color!)

![image](https://user-cdn.hackclub-assets.com/019d446a-fa5e-7273-89e2-4f2cf151984a/20260331_15h55m47s_grim.png)

You dont need pcba considering this is very simple to solder and keeps costs low! All you need to change is the color of your pcb if you want and then you should be ready to order!
